# 2026-LogicLab

2026 리액트 수행평가 프로젝트입니다.

논리 회로 설계 및 시뮬레이션을 위한 웹 기반 교육용 플랫폼
직관적인 UI로 논리 게이트를 조합하여 회로를 설계하고, 실시간으로 동작을 확인할 수 있는 웹 애플리케이션

📌 프로젝트 소개
LogicLab은 학생들을 대상으로 한 논리 회로 학습 플랫폼입니다.

기본 논리 게이트(AND, OR, NOT 등)를 조합하여 복합 회로를 직접 설계하고, 입력값 변화에 따른 출력값을 실시간으로 시뮬레이션할 수 있습니다.

복잡한 회로 설계 도구 없이도 브라우저에서 드래그 앤 드롭으로 쉽게 회로를 구성하고, 결과를 즉시 확인하여 논리 회로의 원리를 효과적으로 학습할 수 있습니다.
