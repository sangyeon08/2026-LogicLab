# 2026-LogicLab

2026 리액트 수행평가 프로젝트입니다.

논리 회로 학습을 위한 웹 기반 교육용 플랫폼

스위치 토글로 게이트를 직접 조작하고, 진리표와 LED 출력을 실시간으로 확인할 수 있는 웹 애플리케이션

## 📌 프로젝트 소개

LogicLab은 학생들을 대상으로 한 논리 회로 학습 플랫폼입니다.

6가지 논리 게이트(AND, OR, NOT, NAND, NOR, XOR)를 선택하고 입력 스위치를 직접 토글하며 출력값과 진리표가 실시간으로 반응하는 것을 확인할 수 있습니다.

자유 시뮬레이터에서 게이트 동작을 탐색하고, 미션을 풀며 실력을 키우고, 오답 데이터를 바탕으로 취약한 부분을 스스로 진단할 수 있습니다.

## 🎯 프로젝트 목표

- 논리 게이트 동작 원리의 시각적 이해
- IoT·컴퓨터 구조 맥락과 연결한 실생활 기반 학습
- 오답 데이터 분석을 통한 자기주도 학습 지원
- 미션 시스템으로 단계별 학습 동기 부여

## ✨ 주요 기능

### 🔬 Sandbox (자유 시뮬레이터)

- 게이트 선택 및 입력 스위치 실시간 토글
- 현재 입력에 해당하는 진리표 행 자동 하이라이트
- 게이트 특성(기호, 동작 규칙, 출력 패턴) 카드 표시
- IoT 센서 라벨 커스텀 (PIR 센서, 온도 센서 등 프리셋)

### 🎯 Challenge (미션 챌린지)

- IoT·스마트팩토리·CPU 연산 시나리오 기반 15개 미션
- 초급(M1~M5) / 중급(M6~M10) / 고급(M11~M15) 단계별 구성
- 고급 미션은 전체 진리표 검증 방식 적용
- 보너스 점수 시스템 (첫 시도 성공 +50점, 힌트 미사용 +30점)

### 📊 Review (학습 분석 다이어리)

- 오답 자동 기록 및 게이트별 정답률 바 그래프
- 취약 게이트 감지 및 해당 미션 바로가기
- 오답 노트 메모 작성 및 복습 완료 표시
- 맞춤형 학습 진단 메시지 제공

## 🗂️ 서비스 구조

```
Sandbox (자유 시뮬레이터)
Challenge (미션 챌린지)
├ 미션 목록 (게이트 필터, 진행도 표시)
└ 미션 상세 (회로 설계, 진리표 검증, 판정)
Review (학습 분석 다이어리)
├ 게이트별 성취도
├ 오답 노트
└ 맞춤형 진단
```

## 📊 시각화 요소

- 스위치 토글 → LED 점등 실시간 연동
- 진리표 현재 행 하이라이트
- 게이트별 정답률 바 그래프
- 미션 전체 진행도 바
